WiiTCPLoad

WiiTCPLoad is an application to send homebrew to your Wii running the Homebrew Channel. It supports both .dol and .elf files and compressing them for a faster transfer rate. You can optionally send arguments to the application when it runs. WiiTCPLoad was originally created by Strikeu, along with modifications for OS X from Tinman, at http://sourceforge.net/projects/wiitcpload/. This version has been updated to support the latest Homebrew Channel.
